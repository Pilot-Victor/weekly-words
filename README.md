# 주간 영어 표현 (Weekly English Expressions)

구글시트 YYMMDD_ 탭에서 **보라색으로 표시된 단어/표현 298개**를 추출해 만든 주간 학습 웹앱입니다.

## 기능
- 매주 **랜덤 3개 표현**(주차 시드 → 두 사람 동일) + 예문 1개
- **월요일 자동 리셋**
- 담당 2명(**규영·병찬**) **하루 1회 완료 체크**
- **주간 누적 실적**(최대 7일/주) 트래킹
- 공유 저장(Firebase) 또는 로컬 저장

## 공유 저장(선택) 설정
`index.html` 상단의 `DB_URL` 에 Firebase Realtime Database URL을 넣으면
규영·병찬의 체크가 실시간으로 공유됩니다. 비워두면 기기별 로컬 저장으로 동작합니다.

- 데이터 원본: `raw_words.csv`
